How to play:
  • Use WASD to move and Z to shoot
  • When you destroy 10 robots you get triple projectile
  • You can write scripts what your robot does run each frame in the field below
    • Script is in JS and is run at the beginning of every frame
    • You can read objects "entities" and "players"; you shouldn't write them - only you'd see that changes
    • Every object in these lists have properties x,y, velocity and direction
    • Entities have also property "resId" which can be 2 or 3: 2 is a neutral/decorational asteroid and 3 is a player's projectile(yours too)
    • You have access to your own direction(in radians) and velocity(0-200) which you can set as you want. They're properties of object "player"
    • To shoot a projectile run function "shoot()" It will shoot the projectile in the direction you're facing and velocity of you + 100, also you have to wait till countdown is <≡ 0
  • To run a script tick "Run code"
  • Scripts examples
Troubleshooting:
  • If you have problems with quicksearch on Firefox, put mouse focus on this textarea and play :)
  • If server is down, please run your own from script GitHub and go to IP version.
